# NEURO-FLOW OS 🧠⚙️

Orquestrador de Produtividade Neural para Times de Engenharia.

## 🚀 Visão Geral
O Neuro-Flow OS é uma plataforma soberana que sincroniza o backlog técnico com a biometria e o estado neural dos desenvolvedores em tempo real, eliminando o burnout e maximizando o "Flow".

## 🛡️ Arquitetura Soberana
Este repositório é auto-contido. Ele utiliza o kernel do **Antigravity** via `libs/llm_integration` para todas as deliberações de alta fidelidade e auditorias neurais.

### Estrutura
- `backend/`: API FastAPI + Ledger SQLModel.
- `frontend/`: Dashboard React Flow.
- `libs/`: Módulos de inteligência e PER (Soberania Local).
- `docs/`: Manifesto V3.1 e Especificações Técnicas.

## 🛠️ Instalação em outro Antigravity
1. Clone este repositório para a pasta `PROJETOS/` do seu Antigravity.
2. Configure o arquivo `.env` com as chaves necessárias (Kimi, Inception, OpenAI).
3. Instale as dependências:
   ```bash
   pip install -r backend/requirements.txt
   npm install --prefix frontend
   ```
4. Inicie o SO:
   ```bash
   python backend/main.py
   ```
