The Cognitive Memory Service on port 8090 is the **long-term ledger** that the ResistOS Kernel consults before every privileged operation: CMS supplies the immutable history (ADR graph, audit trail, SHA-verified backups) so that the kernel can enforce intent-aligned policies, while the kernel in turn tags each syscall with the CMS trace-id it just validated, creating a closed feedback loop where CMS memory becomes the single source of truth that ResistOS uses to decide what is allowed to run, making the kernel “resist” any action whose evidence chain is absent or tampered.
