# 📖 GUIA SIMPLIFICADO: MINHA PRIMEIRA FÁBRICA DE SOFTWARE (T11) 🚀

Olá! Se você está lendo isso, parabéns! Você agora tem o poder de criar aplicativos de alta tecnologia com apenas alguns comandos. Não se preocupe com termos técnicos difíceis, vamos explicar tudo como se fosse um jogo de montar.

---

## 1. O que é a Tecnologia 11 (T11)? 🏗️
Imagine que você tem uma **Fábrica de Brinquedos Mágica**. 
- A **Tecnologia 11** é o cérebro da fábrica.
- Ela sabe como montar um brinquedo perfeito (o aplicativo do cliente) que já vem com manual, peças organizadas e segurança contra quebras.

---

## 2. Como criar um novo Projeto? (O Passo a Passo) 📝

Para criar um novo aplicativo para um cliente, você só precisa me pedir assim no chat:

> *"Antigravity, crie um novo projeto para o cliente **[NOME]** chamado **[PROJETO]**."*

**O que eu vou fazer sozinho:**
1. Criar uma pasta nova em `WORKSPACE/PROJETOS`.
2. Organizar as "caixas" (pastas) do projeto (Clean Architecture).
3. Colocar um "RG" (ID de Linhagem) em cada arquivo para sabermos que ele é original.

---

## 3. O que são as "Sprints" e o "Savepoint"? 🎮

Como em um videogame, você não quer perder o seu progresso, certo?

- **A Sprint**: É como uma "missão" curta. Você decide o que vai fazer hoje (ex: criar a tela de login).
- **O Teste**: Antes de salvar, eu verifico se o que você fez está funcionando.
- **O Savepoint (Commit)**: Se o teste passar, eu salvo o jogo automaticamente para você! Se algo estiver errado, eu te aviso e não salvo, para não estragar o que já estava bom.

---

## 4. Onde ficam as coisas? (Mapa do Tesouro) 🗺️

- **Onde eu crio os aplicativos**: Ficam todos na pasta `WORKSPACE/PROJETOS`.
- **Onde fica a fábrica (NÃO MEXER)**: Fica na pasta `11_PROJECT_FORGE_CODEFORGE` na raiz. É lá que moram os robôs que trabalham para você.
- **Segurança**: Se você tentar salvar algo fora do lugar, eu vou te avisar com um alerta vermelho. É o meu "Guarda de Fronteira" trabalhando.

---

## 5. Dicas de Ouro para Iniciantes ⭐

1. **Sempre peça para eu criar**: Evite criar pastas à mão, peça para mim! Assim eu garanto que a estrutura está "Soberana".
2. **Leia os Relatórios**: Depois que eu terminar uma tarefa, eu sempre te dou um link para um documento. Vale a pena dar uma olhadinha rápida!
3. **Não tenha medo**: Se você errar um comando, o sistema é blindado. Nada de ruim vai acontecer com o cérebro principal (Antigravity).
